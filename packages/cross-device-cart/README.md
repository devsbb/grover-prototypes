# Cross device cart prototype
https://cross-device-cart.web.app

This project was built to check concept about cross device cart management.

### Prerequisites

1. Create **.env.production** and **.env.development**
2. Ask copy [Daniil](github.com/tkach7) 
3. run  ```yarn install```

## Getting Started
run ```yarn start```

## Deployment
Firebase is uses for deployment. First of all, install set up firebase tool.

```
npm install -g firebase-tools
```

Then run 
```
gatsby build
firebase deploy
```


## Built With

* [Gatsby](https://www.gatsbyjs.org/) - The web framework used;
* [Firebase](https://firebase.google.com/?hl=ru) - Database and authentication;
